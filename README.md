Key Features
1. AI Disease Detection

Users can upload a photo of a stray dog, and the system uses a deep learning model to predict possible skin or health conditions.
This helps identify dogs that may need urgent care.

2. Stray Dog Reporting & Live Map

Users can report stray dog sightings or incidents

Each report is pinned on an interactive map

Location, condition, and time are stored for rescue coordination

3. Adoption Platform

Rescue organizations can list dogs available for adoption

Users can browse profiles and request adoption

Admins and shelters can manage adoption requests

4. User Authentication

Secure login and registration
